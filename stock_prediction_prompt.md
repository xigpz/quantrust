## 💡 使用说明

### 变量替换说明
将 `{变量名}` 替换为实际数据，若某项数据无法获取，填写 `暂无数据` 即可，不影响整体分析质量。

### DeepSeek 调用建议配置
```json
{
  "model": "deepseek-reasoner",
  "temperature": 0.3,
  "max_tokens": 4000,
  "top_p": 0.85
}
```
> 使用 `deepseek-reasoner`（R1）模型，开启深度推理，temperature 设低（0.3）保持分析的严谨性，避免过度发散。

### 防止过度乐观/悲观的约束
在 System Prompt 末尾追加以下内容：
```
重要约束：
1. 多空情景概率之和必须等于100%，任何单一情景概率不得超过65%
2. 综合评分禁止出现满分（100分）或零分，极端行情上限90分/下限10分
3. 操作建议中，任何市场环境下激进型仓位不得超过80%，保守型不得低于10%
4. 每个利好判断必须配套列出对应的反驳理由
5. 禁止出现"必涨""必跌""稳赚"等绝对化表述
```
